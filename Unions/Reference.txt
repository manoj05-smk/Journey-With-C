Unions:
   A union is a user-defined data type where all members share the same memory location.
Syntax:
   union Data {
       int i;
       float f;
       char c;
   };
 => To access members of a Union, use the dot (.) syntax.

Structure vs Unions:
* Struct -> Each member has its own memory.
* Union -> All members share the same memory, which means you can only use one of the values at a time.

| Feature | Structure             | Union       |
| ------- | --------------------- | ----------- |
| Memory  | Separate for each     | Shared      |
| Size    | Sum of members        | Max member  |
| Usage   | Store multiple values | Save memory |
